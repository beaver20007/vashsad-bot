- [x] Структура проекта (handlers, services, keyboards)
- [x] /start + главное меню

- [x] Подбор растений (FSM: регион → тип → освещение)

- [x] Форма заявки на услугу → уведомление дизайнеру
- [x] Бриф для индивидуального проекта (FSM, 5 вопросов)
- [x] Лимиты Free-тира (10 чатов / 3 фото / 3 подбора)
- [ ] Заменить in-memory storage на SQLite
- [ ] Настроить BotFather (имя, описание, фото, команды)
- [ ] Задеплоить на VPS

### BotFather настройки
```
Имя:        ВашСад Бот
```

- Бот запущен и работает 24/7

**Цель:** Полный набор инструментов для удержания пользователей

- [ ] FSM-диалог генерации плана участка (параметры → AI-план → PDF)
- [ ] PostgreSQL вместо SQLite
- [ ] Redis для FSM storage
- [ ] Персональный профиль (/profile — регион, участок, мои растения)
- [ ] История запросов пользователя
- [ ] Реферальная программа (пригласи друга → бонус)
- [ ] Портфолио с фото реальных проектов
- [ ] Первый цифровой продукт — PDF-гайд по растениям НО (490 ₽)

**KPI Этапа 1:**
- 500 MAU
- 20+ стартовых услуг/мес
- 50–80 тыс. ₽/мес выручки
