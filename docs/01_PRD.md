# PRD — ВашСад Бот
# Product Requirements Document v1.0

---

## 1. Обзор продукта

**ВашСад Бот** — Telegram-бот дипломированного ландшафтного дизайнера.
Специализация: природный стиль садов, Нижегородская и Владимирская области.

**Миссия:** Сделать профессиональный ландшафтный дизайн доступным через Telegram —
от бесплатной AI-консультации до полного авторского проекта.

**Модель:** Бот — воронка продаж реальных дизайнерских услуг, не самоцель.

---

## 2. Целевая аудитория

### Персона 1 — «Наташа» (дачница)
- 35–55 лет, дача 6–15 соток, Нижний Новгород / Владимир
- Хочет красиво, но не знает с чего начать
- Боль: не понимает какие растения подойдут, боится ошибиться
- Нужно: простой план, подбор растений, советы по уходу

### Персона 2 — «Андрей» (владелец дома)
- 40–60 лет, загородный дом 15–30 соток
- Строит или только купил участок
- Боль: нужен профессионал, но не знает где найти надёжного
- Нужно: полноценный дизайн-проект, доверие к специалисту

### Персона 3 — «Алина» (молодая хозяйка)
- 28–40 лет, коттедж, активна онлайн
- Хочет природный сад «как на Pinterest»
- Нужно: референсы, концепция, конкретные растения

## 3. Функциональные требования

| Функция | Лимит | Приоритет |
|---------|-------|-----------|
| AI-чат по садоводству | 10 сообщений/мес | P0 |
| Фото-диагностика растений | 3 фото/мес | P0 |
| Подбор растений | 3 запроса/мес | P0 |
| Базовый план участка | 1 раз | P1 |
| Просмотр прайс-листа | без лимита | P0 |
| Портфолио дизайнера | без лимита | P1 |

| Услуга | Цена (старт) | Цена (6 мес+) | Срок |
|--------|-------------|---------------|------|

| Сезонный план ухода | 2 500 ₽ | 3 500 ₽ | 2 дня |

### 3.3 Индивидуальный проект (расчёт по брифу)

| Состав | до 10 сот. | 10–20 сот. | 20–50 сот. |
|--------|-----------|-----------|-----------|
| Эскизный | 10 000 ₽ | 18 000 ₽ | 30 000 ₽ |
| Дизайн-проект | 25 000 ₽ | 40 000 ₽ | 65 000 ₽ |
| Рабочий проект | 40 000 ₽ | 60 000 ₽ | 100 000 ₽ |
| Авторский надзор | 3 000 ₽/выезд | 3 000 ₽/выезд | 5 000 ₽/выезд |

### 3.4 Основные команды бота

```
/start        — приветствие + главное меню
/chat         — AI-консультация по садоводству
/plants       — подбор растений (FSM-диалог)
/photo        — диагностика болезни по фото
/plan         — генерация плана участка (FSM)
/price        — прайс-лист услуг
/order        — заказ услуги / бриф проекта
/subscribe    — подписка Сад Про
/portfolio    — портфолио работ
/help         — справка
```

### 3.5 Ключевые User Flows

**Flow 1: Новый пользователь → Первый заказ**
```
/start → Welcome + меню → «Подобрать растения» → FSM (регион, тип, освещение)
→ AI-результат (бесплатно) → «Хотите профессиональный список?»
→ Услуга «Подбор Pro» 1 500 ₽ → Оплата → Уведомление дизайнеру
```

**Flow 2: Диагностика растения**
```
Пользователь присылает фото → Claude Vision анализирует
→ Диагноз + рекомендации → «Нужна помощь специалиста?»
→ Кнопка «Экспресс-консультация» 1 000 ₽
```

**Flow 3: Запрос на проект**
```
/order → «Индивидуальный проект» → FSM-бриф (5 вопросов)
→ Уведомление дизайнеру → Дизайнер отвечает с расчётом → Договор
```

## 4. Нефункциональные требования

| Параметр | Требование |
|----------|-----------|
| Время ответа AI | < 10 сек для текста, < 20 сек для фото |
| Доступность | 99.5% (допустимо 3.6 ч/мес простоя) |
| Хранение данных | Только на сервере дизайнера (приватность) |
| Язык интерфейса | Русский |
| Платформа | Telegram (iOS, Android, Desktop) |
| Авторизация | Telegram ID (без регистрации) |

---

## 5. Критерии готовности MVP

MVP считается готовым когда:
- [ ] Бот отвечает на /start и показывает меню
- [ ] AI-чат работает (Claude API подключён)
- [ ] Фото-диагностика работает (Claude Vision)
- [ ] Подбор растений (FSM) работает
- [ ] Прайс-лист отображается
- [ ] Форма заявки работает → уведомление дизайнеру приходит
- [ ] Лимиты Free-тира считаются правильно
- [ ] Бот задеплоен и работает 24/7
